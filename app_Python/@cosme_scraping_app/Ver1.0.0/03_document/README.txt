========================================
@コスメランキング自動取得アプリ（Python版）
========================================

■ 概要
----------------------------------------
このアプリは、@cosme のランキングページから
「商品名」と「商品ページURL」を自動で取得し、
Excelファイル（.xlsx形式）として出力する
Windows向けスクレイピングツールです。

URLを入力して「開始」ボタンを押すだけで、
上位100件の商品情報を簡単に取得できます。

Python の Tkinter を使用した GUI アプリで、
初心者でも操作できるよう簡潔な画面構成になっています。


■ 主な機能
----------------------------------------
- @cosme のランキングページを自動解析
- ランキング商品名・商品URLを抽出
- Xpath 指定部分のみ抽出する正確なスクレイピング
- 自動でExcelファイル（.xlsx）に書き出し
- ファイル名に @cosme のカテゴリ名と実行日時を自動付与
- GUI操作（URL入力・フォルダ選択・実行）
- 実行中はスピナー表示で進捗を可視化


■ 動作環境
----------------------------------------
- Windows 10 以上
- Python 3.10〜3.13（必須）
- インターネット接続が必要


■ 必要なライブラリ
----------------------------------------
以下はアプリ側で自動インストールされます。

- requests
- beautifulsoup4
- pandas
- openpyxl
- lxml


■ 使い方
----------------------------------------
1. フォルダ内の「run_app.bat」をダブルクリック
2. 必要なライブラリが自動インストールされます
3. GUIアプリが起動します
4. 対象の @cosme ランキングURLを入力
5. Excel出力先のフォルダを選択
6. 「開始」ボタンを押すと取得開始
7. 完了後、自動でExcelが出力されます


■ 出力ファイル名
----------------------------------------
@コスメ_＜カテゴリ名＞_＜実行年月日時分＞.xlsx


■ 注意点
----------------------------------------
・このアプリは個人利用・学習目的で作成したものです。
・@cosme の仕様変更があると動作しなくなる可能性があります。
・大量アクセスを防ぐため、節度ある利用をお願いします。
